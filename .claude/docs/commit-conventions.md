# Commit Conventions

Format: `<type>(<scope>): <short description>`

**Types:** `feat`, `fix`, `refactor`, `chore`, `docs`

**Scopes:** `nvim`, `zsh`, `i3`, `tmux`, `nixos`, `alacritty`, `wezterm`, `kak`, `polybar`

```
feat(nvim): add oil.nvim for file navigation
fix(i3): correct workspace switch keybind
chore(wallpaper): add new city wallpaper
refactor(zsh): consolidate path exports
```
